# 📊 Operations Analytics | KPI & Performance Dashboard (Excel & Google Sheets)


An interactive Excel dashboard designed to analyze call centre performance across **representatives, locations, and customer satisfaction metrics**.  
Built using **Pivot Tables, Charts, and Slicers** with a clean, minimal design for professional reporting.

## ✨ Features
- **Key Metrics (KPIs):** Total Calls, Duration, Revenue, and Customer Ratings.
- **Trend Analysis:** Monthly call volume and revenue tracking.
- **Performance Breakdown:**  
  - By **Representative** (calls handled, satisfaction rating).  
  - By **City** (distribution of calls and results).  
- **Interactive Slicers:** Filter by time, rep, or location to drill into insights.
- **Professional Design:** Hidden gridlines, consistent theme, and clear layout.
